| 编号 | 层 | 位置 | 描述 |
|------|----|------|------|
| P0-A | 服务层 | `internal/mcp/jsonrpc.go` `call()` | 并发数据竞争：多协程直接调用 `transport.Recv()`，stdio bufio.Reader 无读互斥，消息边界损坏 |
| P0-B | 核心层 | `internal/engine/engine.go` | `e.abortFn` 无互斥锁，goroutine 和主循环并发读写，存在 data race |
| P0-C | CLI 层 | `internal/bootstrap/` + `cmd/claude/` | 零测试（TestMain 缺失、集成测试框架未搭建） |
| P0-D | CLI 层 | `internal/bootstrap/auth.go` | 使用 `os.Setenv("ANTHROPIC_API_KEY", ...)` 污染进程环境（测试隔离违规） |
| P0-E | CLI 层 | `internal/bootstrap/auth.go` | 用 `os.Exit(1)` 代替 `return error`，bootstrap 函数不可测试、不可复用 |
| P0-F | CLI 层 | `cmd/claude/main.go` `mcp serve` | 子命令未实现（返回"not implemented"），CLI 层核心功能缺失 |

| P0-H | 核心层 | `internal/compact/micro.go` | `MicroCompactor.Compact` 方法签名与 `Compressor` 接口不匹配，接口多态注入失效 |
